# reactjs-web

This is a starter template for web [reactjs](https://reactjs.org) projects.

## How to use this template

Required:

-   nodejs

```bash
# init project
$ git clone https://github.com/Dpezz/test_keiro_web.git
$ cd test_keiro_web
$ npm install
$ npm start
```

start server [http://localhost:3000](http://localhost:3000)
